# Rakshit Mangal - Portfolio Website\
\
This repository hosts my personal portfolio website built for GitHub Pages.  \
It presents my professional background, technical skills, and selected projects in a clean terminal-inspired, minimal static design.\
\
## Overview\
\
The website is a single-page portfolio focused on:\
- Professional experience (Amazon, Paytm)\
- Applied ML/GenAI and backend engineering projects\
- Technical skill categories\
- Education and publication highlights\
- Contact links for professional outreach\
\
The site is intentionally lightweight, responsive, and GitHub Pages friendly (no framework required).\
\
## What This Project Includes\
\
- `index.html`  \
  Main static portfolio page containing HTML, CSS, and JavaScript for:\
  - Hero/About\
  - Experience\
  - Projects\
  - Skills\
  - Education\
  - Publications\
  - Contact\
\
## Design and UX Direction\
\
- Modern minimal layout with terminal/glass aesthetic\
- Cohesive light color palette with readable contrast\
- Responsive behavior for desktop and mobile\
- Lightweight visual effects and smooth scrolling\
- Theme-consistent header behavior on scroll\
\
## Project Content Highlights\
\
- **Experience:** Resume-aligned achievements for Amazon and Paytm\
- **Projects:** New resume projects appear first, followed by additional projects and legacy entries\
- **Skills:** Backend/Cloud/DevOps, Data/Observability, and ML/GenAI grouped clearly\
- **Education:** Updated coursework based on latest resume\
- **Publication:** Security/privacy and blockchain-in-healthcare aligned wording\
\
